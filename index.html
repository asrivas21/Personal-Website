smart-meeting-summarizer

Smart Meeting Summarizer

A full-stack meeting platform built with a React, TypeScript, and Tailwind frontend on top of a FastAPI backend. Audio is transcribed end-to-end through OpenAI Whisper and summarized with GPT-4, producing clean meeting notes and highlights.

The backend is containerized with Docker Compose and backed by PostgreSQL for structured data, MinIO for raw audio storage, and Redis for async processing across eight REST API endpoints. Users can track action items with assignees, due dates, and status, and export them directly to CSV or email.

autonomous-ai-assistant

Autonomous AI Assistant

An autonomous research agent built on LangChain and the OpenAI API. It decomposes open-ended queries into structured sub-tasks and emits JSON reports, cutting manual research time by roughly 60% on the tasks I benchmarked it against.

Under the hood it runs a RAG pipeline with PDF ingestion via PyMuPDF, semantic chunking, and a Chroma vector database using OpenAI embeddings for sub-second similarity retrieval. The arXiv API is wired in for academic paper ingestion, with end-to-end citation tracking and Pydantic schema validation to keep outputs structured and hallucination-resistant.

f1-podium-predictor

F1 Podium Predictor

An end-to-end machine learning pipeline that predicts Formula 1 podium finishes using Random Forest and KNN models trained on telemetry from the Fast-F1 API. The models hit an R² above 0.7 and about 80% season accuracy on held-out seasons.

Features are engineered from driver consistency, team performance, and historical podium rates, with cross-validation and hyperparameter tuning behind the scenes. An interactive D3.js frontend visualizes predicted versus actual podium counts with season filtering, all served by a Python and Pandas data pipeline.
